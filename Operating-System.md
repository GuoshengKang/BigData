## 快捷键
- F1：Setting Bios
- F12：Entering Boot Menu

## BIOS Setting
- Security-->Secure Boot--> off
- Startup-->Network Boot--> USB HDD
- Startup-->UEFI/Legacy Boot--> Legacy only (或者Both)
- Security-->Virtualization-->Kenel DMA Protection-->off
